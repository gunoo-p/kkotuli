# 끝말잇기 플랫폼

끄투 온라인을 오마쥬한 웹 기반 실시간 한국어 끝말잇기 멀티플레이어 게임 플랫폼
로그인 없이 방 코드 하나로 참여하는 비회원 실시간 게임 서비스

---

## 팀원

| 이름 | 역할 |
|------|------|
| 박건우 | PM · 인프라 설계 및 구축 (GitHub Actions, Docker, AWS EC2) · 실제 코드 구현 및 배포 · 발표 |
| 신재민 | MSA 아키텍처 설계 및 작성 |
| 유상현 | 프로젝트 전체 기획서 설계 및 작성 |
| 김응균 | TDD 기초 설계 · UI/UX 설계 및 기획 |
| 김민성 | 백엔드 기술 설계 · 백엔드 서버 개발 · 문서 작성 |
| 지태웅 | TDD 기초 설계 · 웹소켓 서버 개발 |
| 이찬우 | 자료 정리 · 발표 PPT 작성 |
| 노신용 | DB 설계 · TDD 작성 · TC 작성 |
